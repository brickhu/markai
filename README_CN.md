# 📌 MarkAI — 捕捉 Agent 中的每一个火花。越来越懂你的 AI。

> 🇬🇧 [English version](README.md)

MarkAI 是一个住在你 AI Agent 里的个人知识库。你随口告诉它的事——一个生日、一条链接、一个灵感——它会在你最需要的时刻自动想起来。

---

## 🤔 解决什么问题？

你每天都在跟 AI Agent 对话。但每次对话结束后，它就全忘了。上周问过的生日？没了。想稍后看的那篇文章？找不到了。凌晨两点冒出的灵感？丢了。

**MarkAI 给你的 Agent 装上不会褪色的记忆。**

---

## 💡 30 秒看懂怎么用

```
你:     记住：莉莉的生日是 2019 年 5 月 20 日。
MarkAI: ✅ 已存入。

... 三个月后 ...

你:     莉莉生日哪天？
MarkAI: 5 月 20 日——就剩 2 天了！🎂 需要礼物建议吗？
```

就这样。不用记命令，不用建文件夹。自然说话就行。

---

## 🎯 什么时候用？

| 你说 | 发生什么 |
|------|---------|
| 「记住：比特币 ETF 2024年1月获批」 | 自动提取标题、标签、摘要，存入知识库 |
| 「存一下这个链接 https://...」 | 抓取网页内容，提取关键信息后存储 |
| 「备忘：API Key 是 sk-xxx」 | 安全存储在你的本地机器上（不上传云端） |
| 「React 19 什么时候发布的来着？」 | 搜索你的知识库，即时回答 |
| [任何日常提问] | 自动检查你有没有存过相关内容 |

## 🔒 100% 本地。100% 隐私。

| | MarkAI | 云端方案 |
|---|---|---|
| 数据存在哪 | 你的硬盘 | 别人的服务器 |
| 谁能看到 | 只有你 | 他们的整个工程团队 |
| 离线能用吗 | ✅ 能 | ❌ 不能 |
| 厂商锁定 | 没有——就一个 `.db` 文件 | 绑死在他们平台 |
| 费用 | 永久免费 | API 费、存储费 |

不用注册。不用 API Key。不传遥测。就 Python + SQLite。

---

## 📦 安装

```bash
npx skills add brickhu/markai
```

只需 Python 3（macOS 和 Linux 自带）。零额外依赖。

---

## 🗣️ 触发词

对话中自然说出这些词即可：

| 🇨🇳 | 🇬🇧 | 🇯🇵 | 🇰🇷 |
|-----|-----|-----|-----|
| 记住 · 存一下 · 备忘 | remember · save this · note to self | 覚えて · メモして | 기억해 · 저장해 |

或输入 `/markai` 查看仪表盘。

---

## 🛠 终端管理

```bash
markai stats                    # 看看存了多少
markai search "关键词"           # 全文搜索
markai list                     # 最近条目
markai export --format md       # 导出为 Markdown
```

---

## 📁 你的数据，你说了算

```
~/.agents/skills/markai/data/
└── brain.db    ← 所有知识在这一个文件里
```

**备份**：复制 `brain.db`。**迁移**：移动 `brain.db`。**删除**：删掉文件夹。就这么简单。
